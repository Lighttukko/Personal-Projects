# Personal Projects
## Description
This directory is used to showcase my personal projects.
## Project List
- Trading Strategy Simulator: This program is a backend project designed to help beginner investors test simple trading ideas before risking real money in the market. 
- Personal Website Frontend: This is the code used to build the UI of my personal website, including HTML and CSS.
- Personal Website Backend: This is the code used to connect my personal website to the backend MongoDB database using JavaScript.
- Raspberry PI: A Python spacecraft mini-game using the Raspberry Pi Sense HAT, featuring LED matrix animations, joystick controls, and accelerometer-based interactions.
- AVL Tree: A data structure demonstrating how an AVL Tree is built and balanced in C++.
- Dijkstra's: An implementation of Dijkstra’s shortest path algorithm in C++.
- Red Black Tree: A Java project that visualizes how a Red-Black Tree works through step-by-step operations and structure changes.
